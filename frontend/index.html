Google Sign-In Help

To sign in with a new Google ID, follow these steps:

1. Start Google sign-in Click Continue with Google.
2. Choose another account In the Google popup, click Use another account if your old Gmail is shown first.
3. Enter the new Gmail Type the new Google email and password, then continue.
4. Allow access Approve the Google sign-in request so StudyBot can complete login.
If the popup already opens with the wrong Gmail, first sign out from that Google account in the popup, then choose Use another account. If no popup appears, allow popups for this site and try again.
